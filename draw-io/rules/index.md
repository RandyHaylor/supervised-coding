# Critical Rules

- [fonts.md](fonts.md) — font must be set on BOTH mxGraphModel AND every element style, or PNG export breaks
- [z-order.md](z-order.md) — edges BEFORE vertices in XML, or arrows render in front of boxes
- [label-spacing.md](label-spacing.md) — labels must be 20px+ from arrow lines to avoid overlap
- [japanese-text.md](japanese-text.md) — 30-40px width per Japanese character
- [export.md](export.md) — PNG export CLI commands for Linux and macOS
